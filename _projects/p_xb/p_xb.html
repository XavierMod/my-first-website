xavierbardaji.com

~dt18xxb.com is my first professional web project. It has been build from scratch using HTML, CSS and some JavaScript elements. The first idea was to create a digital portfolio based on minimalist aesthetics. There has been two different versions before the last and current one.

It is important to mention that the current version of this project was developed entirely be me (using some Javascript and css examples and influences) with only one month of coding experience.

In the end, the first version was discarted as it required higher coding skills that I actually had. The next version needed to be cleaner, simpler and easier.

Once the high-fidelity prototype of the second version was finished, I began the coding process. While still working on an early stage, a yellow box was put into the screen to visualize the borders of the header. This produced an unexpected shift: the design pivoted into a third and last version that was going to be visually simpler but better at the same time.
